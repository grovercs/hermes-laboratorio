# ARA Social Lab

Entorno documental de pruebas para ARA Social.

## Objetivo

Probar casos ficticios de redes sociales antes de conectar cuentas reales o trabajar con publicaciones reales.

## Reglas

- No conectar cuentas reales.
- No publicar contenido.
- No responder reseñas reales.
- No usar tokens, cookies, contraseñas ni claves API.
- No tocar la instancia principal de Hermes.
- Trabajar solo con casos ficticios o textos proporcionados explícitamente por Grover.
- Entregar siempre respuestas como borrador para revisión humana.
